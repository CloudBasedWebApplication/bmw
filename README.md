# BMW Cloud Web App

Course project for a cloud-based web application built with a microservices architecture.

## Planned Components

- `apps/web`: unified frontend
- `services/configurator`: car configuration lookup, image mapping, pricing
- `services/merchandise`: merchandise catalog
- `services/road`: route planning and map integration
- `services/ai-feature`: Gemini-backed recommendation flow
- `services/cart`: unified cart state
- `infrastructure`: local MySQL, Redis, and related setup
- `assets`: pre-generated configurator images and metadata
- `docs`: requirements, architecture, and API notes

## Local Development

The project is intended to run locally with Docker during the first development phase.
